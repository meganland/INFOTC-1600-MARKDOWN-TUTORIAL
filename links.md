# Links

Linked texts are simple. The syntax is as follows 

`[link text](examaplelink.com)`

Exmaple: `[link to CNN News](https://www.cnn.com/)`

Outputs

[link to CNN News](https://www.cnn.com/)

Note: You can use this syntax to link between Markdown pages as well!
