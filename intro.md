# What is Markdown?
Markdown is one of many markup languages you can use to add formatting features to plain text documents. The markup language was created by John Gruber in 2004. 
# Why learn Markdown?
Markdown is one of the world’s most popular markup languages. It is easily converted into other formatted markup languages, such as HTML. It is also used on popular websites such as GitHub.
# First things first
To create a Markdown page in GitHub, first create your repository. Then, click “Add File” > “Create new file”. You can name your file whatever you desire, but include “.md” after it. 

Example: “file.md”
